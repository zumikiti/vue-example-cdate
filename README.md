# vue example cdate
[cdate](https://github.com/kawanet/cdate) を試してみた。

とりあえず、カレンダ表示と当日を赤字にしてみた。

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
